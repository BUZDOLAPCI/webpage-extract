# Dedalus Marketplace Info

## Short Description
Extract readable content, tables, and metadata from webpages

## Long Description
Fetch any URL and extract clean markdown content, HTML tables as structured data, and metadata including Open Graph, JSON-LD, and canonical URLs. No headless browser needed. Tools: fetch_url, extract_readable_markdown, extract_tables, extract_metadata.

## Use Cases
- Search
- RAG
- Files
